온도 구성
네이버 검색량 최대 35°
네이버 검색 API로 수집한 검색량을 카테고리 내 최대값 기준으로 정규화. 전시·클래식도 자기 장르 안에서 공정하게 평가받는 핵심 지표입니다.
📡 출처: 네이버 DataLab / 검색 API (주 1회 이상 갱신)
예매순위 최대 35°
KOPIS 박스오피스 순위 기반. 1위=35°, 2위=33°… 순으로 감소. 전시·클래식 등 예매 데이터가 없는 장르는 검색량 비례 최대 15°로 공정하게 보정됩니다.
📡 출처: KOPIS 박스오피스 순위 (주 1회 이상 갱신)
사용자 반응 최대 30°
조회(12°)·좋아요(12°)·저장(6°) 합산. 카테고리 내 상대 정규화로 전시·클래식도 자기 장르 안에서 공정하게 반영. 서비스가 활성화될수록 신뢰도가 높아지는 핵심 지표입니다.
📐 기준: 컬처온도 앱 내 사용자 행동 데이터
마감 임박 최대 12°
D-0(오늘마감)=12°, D-3이내=8°, D-7이내=4°. 기존 대비 대폭 축소해 "곧 끝나는 것 = 인기"로 왜곡되던 문제를 해결했습니다.
📐 기준: 공연 종료일 기준 D-Day 계산
희소성 최대 7°
전체 공연기간이 짧을수록 높음. 3일 이내=7°, 7일 이내=5°, 14일 이내=2°. 마감임박 점수를 동시에 받는 경우 절반만 부여해 단기 공연의 중복 boost를 차단합니다.
📐 기준: 공연 시작일~종료일 전체 기간
신규 오픈 최대 6°
등록 후 3일 이내=6°, 7일 이내=3°. 기존 대비 축소해 "막 열었다"는 이유만으로 과도하게 상위 노출되는 왜곡을 방지합니다.
📐 기준: 컬처온도 DB 최초 등록일 기준
무료입장 5° 보너스
무료 공연·전시에 부여하는 접근성 보너스. 가격 부담 없이 즐길 수 있는 문화행사가 소외되지 않도록 소폭 우대합니다.
📐 기준: 가격 정보 "무료" 포함 또는 관리자 설정
온도는 매일 자동 갱신되며, 예매순위·검색량은 주 1회 이상 업데이트됩니다.
